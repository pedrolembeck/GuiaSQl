# Oracle SQL Guide

Um guia prático de Oracle SQL com exemplos organizados por assunto.

O objetivo deste repositório é servir como material de consulta para estudantes, desenvolvedores e profissionais que desejam revisar rapidamente a sintaxe e os principais recursos do Oracle Database.

---

## Conteúdo

### Fundamentos

* DDL (Data Definition Language)
* DML (Data Manipulation Language)
* DQL (Data Query Language)
* TCL (Transaction Control Language)
* DCL (Data Control Language)

### Consultas

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* DISTINCT

### Relacionamentos

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN
* SELF JOIN

### Funções

* Funções de Texto
* Funções Numéricas
* Funções de Data
* Funções de Conversão
* Funções Analíticas

### Objetos do Banco

* Views
* Sequences
* Indexes
* Synonyms

### Programação PL/SQL

* Procedures
* Functions
* Packages
* Triggers
* Exceptions
* Cursors

### Consultas Avançadas

* Subqueries
* Common Table Expressions (CTE)
* MERGE
* PIVOT / UNPIVOT
* Window Functions
